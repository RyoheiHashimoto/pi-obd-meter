# 算出ロジック・閾値一覧

メーター表示・インジケーター・自動制御に関わるすべての計算式と閾値をまとめる。

> **凡例**
>
> - **config** = `configs/config.json` で車種ごとに設定可能
> - **ハードコード** = ソースコードに直接記述。変更するにはビルドが必要
> - 🚗 = 車種によってチューニングが必要な可能性がある値

---

## 1. 瞬間燃費推定

**ファイル:** `cmd/pi-obd-meter/fuel.go`

### 物理定数（ハードコード・車種非依存）

| 定数 | 値 | 説明 |
|---|---|---|
| 理論空燃比 | 14.7 | ガソリン車共通 |
| ガソリン密度 | 750 g/L | 標準的なレギュラーガソリン |
| 空気密度 | 1.225 g/L | 標準大気 (15°C, 1atm) |

### 🚗 車種チューニング値

| パラメータ | 現在値 | 設定方法 | 備考 |
|---|---|---|---|
| エンジン排気量 | 1.3 L | **config** (`engine_displacement_l`) | 燃費推定の基本パラメータ |
| アイドル燃料消費量 | 0.6 × 排気量 L/h | **排気量から導出** (`idleFuelRateCoeff * displacementL`) | 1.3L→0.78, 2.0L→1.2 L/h |
| エンブレ負荷閾値 | 5.0% | **ハードコード** (`load < 5.0`) | 走行中の極低負荷を燃料カットと判定 |
| 燃費表示最低速度 | 10.0 km/h | **ハードコード** (`minDisplaySpeedKm`) | これ以下ではノイジーなため非表示 |
| 燃費表示上限 | 99.9 km/L | **ハードコード** (`maxDisplayKmL`) | 表示桁数の制限 |

### 計算フロー

```
┌─ MAFセンサーあり ──→  fuelRate = MAF(g/s) × 3600 / (14.7 × 750)  [L/h]
│
├─ MAFセンサーなし ──→  airFlow = (RPM / 2) × (負荷% / 100) × 排気量L / 60  [L/s]
│                        fuelRate = airFlow × 1.225 × 3600 / (14.7 × 750)  [L/h]
│
├─ RPM < 100 or 負荷 < 0.1% → fuelRate = 0.8 L/h（アイドル最低値）
│
└─ 燃費 = 速度(km/h) / fuelRate(L/h)  [km/L]
```

### 表示判定

| 条件 | 表示 | 戻り値 |
|---|---|---|
| speed < 0.5 かつ rpm < 100 | エンジン停止 | (0, 0) |
| fuelRate < 0.01 | 燃料カット | (-1, rate) |
| speed ≥ 10 かつ load < 5% | エンブレ | (-1, rate) |
| speed < 10 | 低速非表示 | (0, rate) |
| 正常 | 燃費表示 | (km/L, rate) |

## 2. 速度ゲージ

**ファイル:** `web/static/js/gauge.js`

### アーク構成

- 角度: 270°アーク（-135° → +135°、7時半～4時半の位置）
- 🚗 最大値: **config** (`max_speed_kmh`, デフォルト 180)
- 目盛り: 9分割 × 5小目盛り = 45刻み（0, 20, 40, ... 180）

### 補間アニメーション（ハードコード）

| パラメータ | 値 | 説明 |
|---|---|---|
| LERP係数 | 0.15 | 毎フレームで差分の15%を追従 |
| スナップ閾値 | 0.05 | これ以下で目標値に即収束 |
| 停止閾値 | 0.01 | これ以下でアニメーション停止 |

### 🚗 速度帯カラー（ハードコード）

| 速度帯 | 色 | Hex |
|---|---|---|
| ≥ 120 km/h | 赤 | `#f44336` |
| ≥ 100 km/h | 橙 | `#ff9800` |
| ≥ 80 km/h | 黄 | `#ffeb3b` |
| ≥ 60 km/h | 薄緑 | `#69f0ae` |
| ≥ 30 km/h | 青 | `#42a5f5` |
| < 30 km/h | 灰青 | `#78909c` |

> 法定速度に基づくが、車種・用途によって閾値を変えたい場合がある。
> 例: 軽自動車なら 100 km/h 以上を赤にする等。

## 3. スロットルアーク

**ファイル:** `web/static/js/gauge.js`

### 🚗 アイドル開度ベースライン

**config** (`throttle_idle_pct`, デフォルト 11.5)

> **車種依存度: 高。** ECU によってアイドル時の報告値が異なる。
> DYデミオは 11-12% 程度だが、車種によって 5-20% まで幅がある。
> config.json で車種ごとに設定可能。

生のスロットル値からベースラインを引いてゼロ始まりに正規化:

```
表示% = max(0, 生OBD値% − 11.5)
```

### 色（ハードコード・車種非依存）

HSL色相が開度に応じて線形変化:

```
色相 = 210° × (1 − 開度% / 100)
```

| 開度 | 色相 | 色 |
|---|---|---|
| 0% | 210° | 青（巡航） |
| 50% | 105° | 緑 |
| 100% | 0° | 赤（全開） |

### 補間（ハードコード）

LERP係数 = 0.2（速度ゲージの 0.15 より速い追従）

## 4. ECO インジケーター

**ファイル:** `web/static/js/indicators.js`, `cmd/pi-obd-meter/api.go`

### 🚗 閾値

**通常走行 (≥ 30 km/h) — km/L 判定（排気量から導出）:**

| 色 | 導出式 | 1.3L での値 | 2.0L での値 |
|---|---|---|---|
| 緑 | ≥ `round(20 / 排気量, 1)` km/L | ≥ 15.4 | ≥ 10.0 |
| 橙 | ≥ `round(13 / 排気量, 1)` km/L | ≥ 10.0 | ≥ 6.5 |
| 赤 | 上記未満 | < 10.0 | < 6.5 |

> 排気量が大きいほど燃費が悪いため、閾値を自動で下げる。
> `/api/config` で算出してフロントエンドに渡す。

**低速 (< 30 km/h) — L/h 判定（排気量に連動）:**

| 色 | 条件 | 1.3L での値 |
|---|---|---|
| 緑 | < 1.5 × 排気量 L/h | < 1.95 L/h |
| 橙 | < 3.0 × 排気量 L/h | < 3.90 L/h |
| 赤 | ≥ 3.0 × 排気量 L/h | ≥ 3.90 L/h |

> 排気量係数 1.5 / 3.0 は **ハードコード** (`api.go` L70-71)。
> 排気量は **config** (`engine_displacement_l`) から取得するため、
> 他車種でも排気量を設定すれば自動で閾値が変わる。

**特殊状態:**

| 状態 | 色 | 表示 |
|---|---|---|
| エンブレ / 燃料カット | 緑 | 給油間平均燃費 + ▸ |
| 停車中 + 平均燃費あり | 平均燃費の km/L 判定 | 同上 |
| 異常高値 (> 30 km/L) | 緑 | 給油間平均燃費 + ▸ |

### 低速判定の速度閾値（ハードコード）

```javascript
ECO_LOW_SPEED_THRESHOLD = 30  // km/h
```

## 5. 水温インジケーター

**ファイル:** `web/static/js/indicators.js`

### 🚗 温度閾値（ハードコード）

| 温度 | 色 | 意味 |
|---|---|---|
| データなし (≤ 0°C) | 消灯 | — |
| 1 - 69°C | 橙 | 暖機中 |
| 70 - 104°C | 緑 | 正常動作温度 |
| ≥ 105°C | 赤 | 過熱警告 |

> 一般的なサーモスタット開弁温度（80-90°C）を考慮した汎用的な閾値。
> ほとんどの車種でそのまま使えるが、サーモスタット温度が異なる車種では
> 暖機中の橙が長く点灯する可能性がある。

## 6. トリップインジケーター

**ファイル:** `web/static/js/indicators.js`

### 🚗 閾値（タンク容量 × 排気量から導出）

| 距離 | 色 | 導出式 | 1.3L/40L での値 |
|---|---|---|---|
| < 警告 | 緑 | `tank × ecoGreen × 0.5` | < 308 km |
| 警告 - 危険 | 橙 | `tank × ecoGreen × 0.85` | 308 - 523 km |
| > 危険 | 赤 | | > 523 km |

> `ecoGreen` = ECO 緑閾値 (km/L)。排気量とタンク容量から航続距離を推定。
> タンク容量は **config** (`fuel_tank_l`, デフォルト 40L)。

## 9. メンテナンスリマインダー

**ファイル:** `internal/maintenance/manager.go`

### 🚗 デフォルト項目（ハードコード、config で上書き可能）

| 項目 | 種別 | 間隔 | 警告 | 備考 |
|---|---|---|---|---|
| エンジンオイル交換 | 距離 | 3,000 km | 80% (2,400 km) | シビアコンディション基準 |
| エアフィルター交換 | 距離 | 20,000 km | 85% (17,000 km) | |
| タイヤローテーション | 距離 | 10,000 km | 80% (8,000 km) | |
| 車検 | 日付 | 730日 | 90% (657日) | 自家用車2年ごと |
| ATF交換 | 距離 | 40,000 km | 90% (36,000 km) | マツダ推奨は4万km |

`configs/config.json` の `maintenance_reminders` 配列で項目を自由に追加・変更可能。

### 計算式

**距離ベース:**

```
進捗 = (現在総走行距離 − 最終リセット時距離) / 間隔km
警告 = 進捗 ≥ 警告閾値
期限切れ = 進捗 ≥ 1.0
```

**日付ベース:**

```
進捗 = 経過日数 / 間隔日数
警告 = 進捗 ≥ 警告閾値
期限切れ = 進捗 ≥ 1.0
```

同日再通知なし（1日1回まで）。

## 10. トリップ積算

**ファイル:** `internal/trip/tracker.go`

### 計算式（車種非依存）

| 項目 | 計算式 |
|---|---|
| 距離 Δ | `(speedKmh / 3600) × Δt秒` を毎サイクル加算 |
| 燃料消費 Δ | `(fuelRateLH / 3600) × Δt秒`（エンブレ時は 0） |
| 平均燃費 | `距離km / 燃料消費L`（0.05L 未満は非表示） |
| 平均速度 | `距離km / 走行時間秒 × 3600` |
| 走行時間 | speed > 1.0 km/h の累積秒数 |
| アイドル時間 | speed ≤ 1.0 km/h の累積秒数 |

- 10秒超のギャップはデータ欠損とみなしスキップ
- 状態永続化: 60サンプルごとにアトミック書き込み

## 11. OBD ポーリング

**ファイル:** `internal/obd/pids.go`, `cmd/pi-obd-meter/main.go`

### PID パース式（OBD-2 標準・車種非依存）

| PID | 計算式 | 単位 |
|---|---|---|
| 0x0C RPM | `(A << 8 \| B) / 4` | rpm |
| 0x0D 車速 | `A` | km/h |
| 0x04 負荷 | `A × 100 / 255` | % |
| 0x11 スロットル | `A × 100 / 255` | % |
| 0x05 水温 | `A − 40` | °C |
| 0x10 MAF | `(A << 8 \| B) / 100` | g/s |

### 🚗 ポーリング間隔

| 設定 | 値 | 設定方法 |
|---|---|---|
| 高速ポーリング | 150 ms | **config** (`poll_interval_ms`, 未指定時 150) |
| 全PID | 高速 × 5 = 750 ms | ハードコード (`fullEveryN = 5`) |
| 🚗 OBDプロトコル | "6" (CAN 11bit 500k) | **config** (`obd_protocol`) |

## 車種チューニングまとめ

他車種に適用する際にチェックが必要な項目の一覧。

### config.json で対応（ビルド不要）

| 項目 | パラメータ | 例 |
|---|---|---|
| エンジン排気量 | `engine_displacement_l` | 1.5, 2.0 等 |
| スロットルアイドル開度 | `throttle_idle_pct` | 11.5 (DYデミオ), 8.0 (軽), 15.0 (大排気量) |
| 燃料タンク容量 | `fuel_tank_l` | 40 (DYデミオ), 60 (セダン) |
| メーター最大速度 | `max_speed_kmh` | 180, 260 等 |
| OBDプロトコル | `obd_protocol` | "0"=自動, "6"=CAN 11bit |
| ポーリング間隔 | `poll_interval_ms` | ECU応答速度に依存 |
| メンテ項目 | `maintenance_reminders` | 車種ごとの推奨間隔 |

### 排気量・タンク容量から自動導出

| 項目 | 導出式 | 1.3L/40L での値 |
|---|---|---|
| アイドル燃料消費 | `0.6 × 排気量` L/h | 0.78 L/h |
| ECO 緑閾値 (km/L) | `round(20 / 排気量, 1)` | 15.4 km/L |
| ECO 橙閾値 (km/L) | `round(13 / 排気量, 1)` | 10.0 km/L |
| トリップ警告 | `tank × ecoGreen × 0.5` | 308 km |
| トリップ危険 | `tank × ecoGreen × 0.85` | 523 km |

### ハードコード（ソース変更 + ビルドが必要）

| 項目 | ファイル | 現在値 | 影響度 |
|---|---|---|---|
| ECO 低速閾値 (速度) | `indicators.js` | 30 km/h | 🟢 ほぼ汎用 |
| ECO 排気量係数 (L/h) | `api.go` | 1.5 / 3.0 | 🟢 排気量に連動 |
| 速度帯カラー閾値 | `gauge.js` | 120/100/80/60/30 | 🟡 用途による |
| 水温閾値 | `indicators.js` | 70 / 105 | 🟢 ほぼ汎用 |
| エンブレ負荷閾値 | `fuel.go` | 5.0% | 🟢 ほぼ汎用 |
| 輝度スケジュール | `brightness.go` | 上記参照 | 🟢 地域・季節 |

> 🟡 = 推奨調整, 🟢 = そのまま使える可能性が高い

# メーター表示指標ガイド

車載メーター（800×480 LCD）に表示される各指標の意味と、運転中の活用方法。

## ゲージ内アーク

速度ゲージの内側にスロットルアークを1本表示。

### スロットル

- **何を見てるか**: アクセルペダルの踏み込み量（スロットルバルブ開度）。ドライバーの入力そのもの。
- **範囲**: config の `throttle_idle_pct` を 0%、`throttle_max_pct` を 100% として正規化
- **使い方**: アクセル操作の可視化。踏み込み量と加速の対応を客観的に確認できる。

## 右パネル インジケーター

速度ゲージの右に縦に並ぶステータス表示。ドット色（緑/橙/赤）で状態を示す。

### ECO — 燃費

- **表示値**:
  - 走行中: 瞬間燃費 (km/L)
  - 停車/エンブレ: 給油間平均燃費 + ▸ マーク
- **ドット色**: 緑 (≥15 km/L) / 橙 (≥10 km/L) / 赤 (<10 km/L)
- **使い方**: 総合的な燃費の良し悪し。最終的な答え。
  - 普段は色だけ見ればOK（緑なら効率的な運転）
  - ギア変更後に km/L が改善したか確認

### TRIP — 走行距離

- **表示値**: 前回給油からの走行距離 (km)
- **ドット色**: 緑 → 橙 → 赤（給油タイミングの目安）
- **使い方**: ガソリン残量の目安。

### TEMP — 冷却水温

- **表示値**: 冷却水温 (°C)
- **ドット色**: 橙 (<70°C 暖機中) / 緑 (70–105°C 正常) / 赤 (>105°C 過熱)
- **使い方**: 始動直後の暖機完了確認。105°C 超えは即対処が必要。

### MAINT — メンテナンス

- **表示値**: アラート件数
- **ドット色**: 緑 (0件) / 橙 (注意あり) / 赤 (期限超過あり)
- **使い方**: オイル交換等のメンテナンス時期を通知。

### WiFi / OBD — 接続状態

- **表示値**: OK / NG
- **ドット色**: 緑 (接続) / 赤 (未接続)
- **使い方**: 通信状態の確認。WiFi は GAS データ送信に、OBD はエンジンデータ取得に必要。

## 運転シナリオ別ガイド

### 普段の運転

ECO の色だけ見る。緑ならOK。

### 信号待ち → 発進

穏やかなアクセルでスロットルアークの跳ね上がりを抑える。

## 表示しない指標

| 指標 | 非表示の理由 |
|---|---|
| RPM | ReadFast を2PIDに絞るため削除。ReadAll（2.5秒間隔）でのみ取得 |
| kPa（インマニ圧）アーク | 同上。ELM327マルチPID非対応で表示更新が追いつかない (Issue #54) |
| L/h（燃料消費レート）アーク | 同上 |
| エンジン負荷 (%) | kPa とほぼ同じ動きをする。独自の有用情報がない |
| 推定トルク / 馬力 | OBD データからの推定精度が ±30%。不正確な数値は判断を誤らせる |
| ギア段表示 | トルコン滑りで RPM/速度比率が不安定。リニアに表示できない |

## OBDデータ更新頻度

| 項目 | 更新間隔 | 読み取り元 |
|---|---|---|
| 速度 | ~200ms | ReadFast (毎サイクル) |
| スロットル | ~200ms | ReadFast (毎サイクル) |
| ECO (燃費) | ~1秒 | ReadAll (5サイクルに1回) で再計算 |
| TRIP | ~200ms | 車速積分 (毎サイクル) |
| TEMP (水温) | ~1秒 | ReadAll (5サイクルに1回) |
| RPM / 負荷 / MAP / MAF | ~1秒 | ReadAll (内部計算用、非表示) |

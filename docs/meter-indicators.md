# メーター表示指標ガイド

車載メーター（800×480 LCD）に表示される各指標の意味と、運転中の活用方法。

## ゲージ内アーク（3本の円弧）

速度ゲージの内側に3本のアークを表示。エンジンの状態をリアルタイムに可視化する。

### kPa（インマニ圧 / MAP）— 中間アーク

- **何を見てるか**: インテークマニホールドの絶対圧力。エンジンが実際に吸い込んでいる空気の量。
- **範囲**: アイドル時 ~30 kPa → 全開時 ~101 kPa（大気圧 = NA車の上限）
- **使い方**: スロットルアークとの比較で「踏んでも効かない領域」を発見する。
  - スロットルを開けていくと kPa も上がる
  - ある時点で kPa が頭打ちになる → これ以上踏んでも吸える空気は増えない
  - その状態でさらに加速したければシフトダウンが有効

### L/h（燃料消費レート）— 内側中間アーク

- **何を見てるか**: 1時間あたりの推定燃料消費量。「今この瞬間、どれだけ燃料が燃えているか」。
- **範囲**: アイドル時 ~0.5–1.0 L/h、巡航 ~3–5 L/h、加速時 ~10+ L/h
- **使い方**: ギア変更の効果を即座に確認する。
  - 4速巡航中の L/h を確認 → 3速に落とす → L/h が下がったら3速のほうが効率的
  - 信号待ちからの発進で、急発進 vs ゆっくり発進の燃料コスト差を体感
  - エアコン ON/OFF の影響も見える（アイドル時の L/h が変わる）

### スロットル — 最内側アーク

- **何を見てるか**: アクセルペダルの踏み込み量（スロットルバルブ開度）。ドライバーの入力そのもの。
- **範囲**: config の `throttle_idle_pct` を 0%、`throttle_max_pct` を 100% として正規化
- **使い方**: kPa との比較で「踏み込みに対するエンジンの応答」を見る。
  - スロットルだけ上がって kPa が追従しない → エンジンの吸気限界に達している
  - 普段は体感でわかるが、アークの動きの差として客観的に確認できる

### アークの組み合わせで見えること

| 比較 | 見えること |
|---|---|
| **kPa vs スロットル** | 踏んでも効かない領域（kPa が頭打ち、スロットルだけ伸びる） |
| **L/h + ギア変更** | シフトダウン/アップが燃料消費にどう影響したか |
| **kPa + RPM（右パネル）** | 高 kPa + 低 RPM = エンジンが苦しい → シフトダウン推奨 |

## 右パネル インジケーター

速度ゲージの右に縦に並ぶステータス表示。ドット色（緑/橙/赤）で状態を示す。

### RPM — エンジン回転数

- **ドット色**: 緑 (<3,000) / 橙 (3,000–4,500) / 赤 (≥4,500)
- **使い方**: HOLD モードでのシフトタイミング判断。
  - kPa が高いのに RPM が低い（2,000以下）→ シフトダウンしたほうが効率的
  - 燃費重視なら 2,000〜2,500 rpm でシフトアップ
  - 加速重視なら 3,000〜3,500 rpm（トルクピーク付近）でシフトアップ

### ECO — 燃費

- **表示値**:
  - 走行中: 瞬間燃費 (km/L)
  - 停車/エンブレ: 給油間平均燃費 + ▸ マーク
- **ドット色**: 緑 (≥15 km/L) / 橙 (≥10 km/L) / 赤 (<10 km/L)
- **使い方**: 総合的な燃費の良し悪し。最終的な答え。
  - 普段は色だけ見ればOK（緑なら効率的な運転）
  - ギア変更後に km/L が改善したか確認

### TRIP — 走行距離

- **表示値**: 前回給油からの走行距離 (km)
- **ドット色**: 緑 → 橙 → 赤（給油タイミングの目安）
- **使い方**: ガソリン残量の目安。

### TEMP — 冷却水温

- **表示値**: 冷却水温 (°C)
- **ドット色**: 橙 (<70°C 暖機中) / 緑 (70–105°C 正常) / 赤 (>105°C 過熱)
- **使い方**: 始動直後の暖機完了確認。105°C 超えは即対処が必要。

### MAINT — メンテナンス

- **表示値**: アラート件数
- **ドット色**: 緑 (0件) / 橙 (注意あり) / 赤 (期限超過あり)
- **使い方**: オイル交換等のメンテナンス時期を通知。

### WiFi / OBD — 接続状態

- **表示値**: OK / NG
- **ドット色**: 緑 (接続) / 赤 (未接続)
- **使い方**: 通信状態の確認。WiFi は GAS データ送信に、OBD はエンジンデータ取得に必要。

## 運転シナリオ別ガイド

### 普段の運転

ECO の色だけ見る。緑ならOK。

### HOLD モードで燃費を探る

1. 同じ速度で4速巡航中、L/h を確認
2. 3速に落とす → L/h が下がったら3速のほうが効率的
3. ECO (km/L) でも改善を確認

### 加速時

1. スロットルを踏んでいく
2. kPa アークが頭打ちになったら → これ以上踏んでも加速は変わらない
3. さらに加速したければシフトダウンで回転を上げる

### 信号待ち → 発進

1. 信号待ち中の L/h を確認（アイドル燃料コスト）
2. 発進時、穏やかなアクセルで L/h の跳ね上がりを抑える

## 表示しない指標

| 指標 | 非表示の理由 |
|---|---|
| エンジン負荷 (%) | kPa とほぼ同じ動きをする。独自の有用情報がない |
| 推定トルク / 馬力 | OBD データからの推定精度が ±30%。不正確な数値は判断を誤らせる |
| ギア段表示 | トルコン滑りで RPM/速度比率が不安定。リニアに表示できない |

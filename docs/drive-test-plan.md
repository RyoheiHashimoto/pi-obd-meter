# 実走テスト計画

## 目的

CAN ギア比 (0x230 B2) のオーバーフロー仮説を実走データで検証し、残課題を解決する。

## 事前準備（Mac で）

```bash
# Pi に SSH して CAN ログ記録スクリプトを準備
ssh laurel@pi-obd-meter.local

# メーターサービスを停止（CAN を占有しないように）
sudo systemctl stop pi-obd-meter

# CAN ログ記録開始（全 ID をダンプ）
candump can0 -t A > /tmp/can_drive_test_$(date +%Y%m%d).txt &
echo "記録開始: PID=$!"
```

## テスト項目

### テスト 1: L HOLD 全開加速（最重要）

**目的**: 1速 TC ロック時に B2 が 282 (オーバーフロー→26) 付近で安定するか確認

**運転操作**:
1. 停車状態から **L レンジ** に入れる
2. **HOLD** を押す（1速固定）
3. アクセルを **ベタ踏み**、3000-4000rpm まで引っ張る
4. **10秒キープ**（TC がロックする時間を与える）
5. アクセルオフ → **エンジンブレーキで減速**
6. 停車

**期待結果**:
- 加速中: B2=25-27（オーバーフロー域、TC スリップ大）
- 高速域で TC ロック: B2 が 250→255→0→25-26 と遷移する可能性
- エンブレ: B2 が理論値付近（250-282 域）で安定

**注意**: 安全な直線道路で実施。後続車に注意。

### テスト 2: S レンジ 1速加速

**目的**: S レンジの 1速で高回転まで引っ張ったときの B2

**運転操作**:
1. **S レンジ** で発進
2. シフトアップせずに **4000rpm くらいまで** 引っ張る（S はシフトアップが遅いので自然にできる）
3. そのまま 2速に上がるのを待つ

**期待結果**: 1速→2速の遷移で B2 が 26→150 付近に変化する過程が見える

### テスト 3: R レンジ走行

**目的**: R で TC ロック時の B2 を確認（理論値 2.648 → B2=9?）

**運転操作**:
1. 駐車場の直線で **R** に入れる
2. **ゆっくりバック**（5 km/h 程度）→ B2 記録
3. 可能なら **少し速めにバック**（10 km/h 程度）→ B2 変化を見る

**期待結果**: 低速では B2=26（TC スリップ）、速度が上がると B2 が 9 付近に？

### テスト 4: 4速→3速ダウンシフト（ロックアップ条件検証）

**目的**: ニューモデル速報の記載「3速で約36km/hまでロックアップ」を検証

**運転操作**:
1. **60 km/h 4速巡航**
2. **アクセルオフ**のまま減速
3. 自然にシフトダウン → B2 の変化を記録

**期待結果**: 4速 B2=73 → 3速 B2=100 の遷移と、ロックアップが外れるタイミング

### テスト 5: HOLD 2速 vs 3速 比較

**目的**: HOLD の TC タイト制御が B2 に反映されるか

**運転操作**:
1. **D HOLD** で 2速固定、30 km/h 巡航 10秒
2. **D HOLD** で 3速固定、40 km/h 巡航 10秒
3. **S レンジ** で 3速、40 km/h 巡航 10秒（比較用）

**期待結果**:
- D HOLD 3速: B2 ≒ 100（タイト、メモリでは avg=100.9）
- S 3速: B2 ≒ 97（スリップ大、メモリでは avg=96.7）

```bash
kill %1

# ファイルサイズ確認
ls -lh /tmp/can_drive_test_*.txt

# Mac にコピー（必ず実行！）
# Mac 側で:
scp laurel@pi-obd-meter.local:/tmp/can_drive_test_*.txt ~/Desktop/
```

## Mac への保存（必須）

**テスト後すぐに Mac にコピーすること。** Pi の /tmp はリブートで消える。

```bash
# Mac のターミナルで実行
scp laurel@pi-obd-meter.local:/tmp/can_drive_test_*.txt ~/Desktop/

# 確認
ls -lh ~/Desktop/can_drive_test_*.txt
wc -l ~/Desktop/can_drive_test_*.txt
```

保存先: `~/Desktop/can_drive_test_YYYYMMDD.txt`
